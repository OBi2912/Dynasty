## Customization

### Themes
Modify CSS custom properties in `:root` and `[data-theme="dark"]` for custom color schemes.

### Languages
Add new languages by extending the `data-*` attributes and updating the `updateLanguage()` function.

### Products
Add new products by following the existing HTML structure and updating the filtering logic.
